   a) Export Zod schemas (one per source):
      ```typescript
      export const pickupLogSchema = z.object({
        location_name: z.string().min(1, 'Location name is required'),
        pallet_count: z.coerce.number().int().positive('Pallet count must be positive'),
        preferred_date: z.coerce.date({ errorMap: () => ({ message: 'Invalid date for preferred_date' }) }),
        status: z.enum(['submitted','confirmed','transport_booked','picked_up','at_warehouse','in_outbound_shipment','in_transit','delivered','intake_registered','cancelled']).default('delivered'),
        notes: z.string().optional(),
      })
      ```

      - `intakeLogSchema`: facility_name (string min 1), staff_name (string min 1), delivery_date (coerce date), origin_market (string optional), notes (string optional)
      - `greenloopSchema`: facility_name (string min 1), staff_name (string min 1), activity_type (enum wash/pack), product_name (string min 1), report_date (coerce date), quantity (coerce number int positive), notes (string optional)
      - `invoiceBinderSchema`: intake_reference (string min 1), invoice_status (enum not_invoiced/invoiced/paid), invoice_number (string optional), invoice_date (coerce date optional), notes (string optional)
      - `transportCostSchema`: pickup_reference (string min 1), provider_name (string min 1), transport_type (enum direct/consolidation), transport_cost_eur (coerce number positive)

   b) Export `getSchemaForSource(sourceId: ImportSourceId)` that returns the correct Zod schema.

   c) Export `validateRows<T>(rows: unknown[], schema: z.ZodSchema<T>): { valid: T[]; errors: { row: number; field: string; message: string }[] }`:

      - Unmapped columns are dropped (not carried through)

      - Type `FKLookups = { facilities?: Record<string, string>; products?: Record<string, string>; locations?: Record<string, string>; intakeReferences?: Record<string, string>; pickupReferences?: Record<string, string>; providers?: Record<string, string> }`
      - For each row: if row has facility_name and lookups.facilities exists, replace facility_name with prison_facility_id from lookup. If not found, add to errors with message "Unknown facility: {name}". Same pattern for product_name -> product_id, location_name -> location_id, intake_reference -> intake_record_id, pickup_reference -> pickup_id, provider_name -> transport_provider_id.
      - Return { resolved: rows with UUIDs, errors: rows with unresolvable FKs }
