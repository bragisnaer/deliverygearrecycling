   **getProducts():**
   - Call `auth()` to get session, then `withRLSContext(session.user, async (tx) => { ... })`
   - SELECT from products table ordered by name
   - Return `{ products: Array<{id, name, product_code, product_group, processing_stream, weight_grams, active, created_at}> }`

   **getProduct(id: string):**
   - Validate id with zod uuid schema
   - withRLSContext query: SELECT from products WHERE id = id
   - Also fetch product photos: query storage for list of files at `{tenant_id}/products/{id}/photos/` path using `productsBucket.list()`
   - Return product with photo URLs (generate signed URLs with 1 hour expiry via `productsBucket.createSignedUrl()`)

   **createProduct(formData: FormData):**
   - requireRecoAdmin()
   - Parse and validate with zod: name (string, min 1, max 200), product_code (string, min 1, max 50), product_group (string, optional), processing_stream ('recycling' | 'reuse'), description (string, optional), weight_grams (string, optional — nullable for Clothing)
   - Check product_code uniqueness within tenant: SELECT WHERE tenant_id = session.user.tenant_id AND product_code = data.product_code. If exists, return `{ error: 'Product code already exists for this tenant' }`
   - INSERT into products with tenant_id from session.user.tenant_id
   - revalidatePath('/products')
   - Return `{ success: true, productId: newProduct.id }`

   **updateProduct(id: string, formData: FormData):**
   - requireRecoAdmin()
   - Same zod validation as create
   - Check product_code uniqueness excluding current product: WHERE tenant_id = ... AND product_code = ... AND id != id
   - UPDATE products SET ... WHERE id = id
   - revalidatePath(`/products/${id}`)
   - Return `{ success: true }`

   **uploadProductPhoto(productId: string, formData: FormData):**
   - requireRecoAdmin()
   - Extract file from formData
   - Validate: file.size <= 5 * 1024 * 1024 (5MB), file.type.startsWith('image/')
   - Check existing photo count: list files at product photo path, reject if >= 5
   - Generate filename: `${Date.now()}-${file.name}`
   - Upload via `productsBucket.upload(getProductPhotoPath(tenantId, productId, filename), file)`
   - revalidatePath(`/products/${productId}`)
   - Return `{ success: true }`

   **deleteProductPhoto(productId: string, filePath: string):**
   - requireRecoAdmin()
   - Validate filePath starts with expected tenant prefix
   - Delete via `productsBucket.remove([filePath])`
   - revalidatePath(`/products/${productId}`)
   - Return `{ success: true }`
