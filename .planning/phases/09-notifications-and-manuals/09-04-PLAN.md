Import `auth` from `@/auth`, `db`, `manualPages`, `manualPageVersions`, `withRLSContext` from `@repo/db`, `eq`, `desc`, `and` from `drizzle-orm`, `revalidatePath` from `next/cache`.

Helper to build user object:
```typescript
async function getAdminUser() {
  const session = await auth()
  if (!session?.user || session.user.role !== 'reco-admin') throw new Error('Unauthorized: reco-admin only')
  return { ...session.user, sub: session.user.id! }
}
```

**`getManualPages()`:**
- Call `getAdminUser()`
- Use `withRLSContext(user, async (tx) => { ... })` to SELECT all from `manualPages` ordered by `context ASC, display_order ASC, title ASC`
- Return array of page rows

**`getManualPage(id: string)`:**
- Call `getAdminUser()`
- Use `withRLSContext` to SELECT from `manualPages` WHERE `id = id`, limit 1
- Return single page or null

**`createManualPage(data: { context: 'client' | 'prison'; slug: string; title: string })`:**
- Call `getAdminUser()`
- Validate: slug must be lowercase, alphanumeric with hyphens only (regex: `/^[a-z0-9-]+$/`), throw if invalid
- Use `withRLSContext` to INSERT into `manualPages` with provided data + `updated_by: user.sub`
- Call `revalidatePath('/manual-editor')`
- Return the created page row (use `.returning()`)

**`saveManualPage(id: string, contentMd: string, title: string)`:**
- Call `getAdminUser()`
- Use `withRLSContext(user, async (tx) => { ... })`:
  1. SELECT current content_md from manualPages WHERE id
  2. If current row exists AND current.content_md !== contentMd, INSERT version snapshot into `manualPageVersions` with `manual_page_id: id`, `content_md: current.content_md`, `saved_by: user.sub`
  3. UPDATE `manualPages` SET `content_md = contentMd`, `title = title`, `updated_at = new Date()`, `updated_by = user.sub` WHERE id
- Call `revalidatePath('/manual-editor')`

**`togglePublish(id: string, published: boolean)`:**
- Call `getAdminUser()`
- Use `withRLSContext` to UPDATE `manualPages` SET `published = published`, `updated_at = new Date()` WHERE id
- Call `revalidatePath('/manual-editor')`
- Call `revalidatePath('/manual')` (client-facing)

**`getVersionHistory(manualPageId: string)`:**
- Call `getAdminUser()`
- Use `withRLSContext` to SELECT from `manualPageVersions` WHERE `manual_page_id = manualPageId` ordered by `saved_at DESC`, limit 20
- Return array of version rows

**`deleteManualPage(id: string)`:**
- Call `getAdminUser()`
- Use `withRLSContext` to DELETE from `manualPages` WHERE id (CASCADE deletes versions)
- Call `revalidatePath('/manual-editor')`
