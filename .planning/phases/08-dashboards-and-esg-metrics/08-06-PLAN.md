    **1. Extend `actions.ts`** — add two new functions (keep existing `getAssignedPickups` unchanged):

    **`getTransportProviderInfo()`:**
    Returns `{ provider_type: 'direct' | 'consolidation'; provider_name: string } | null`.
    Query: Get the transport provider linked to the current user. Use `auth()` to get session, then query `transport_providers` where `user_id = session.user.id`. Return provider_type and name.

    **`getWarehouseInventory()`:**
    Returns `{ pickup_id: string; reference: string; client_name: string; product_summary: string; pallet_count: number; arrival_date: string; days_held: number }[]`.
    Only called for consolidation providers. Query pickups with status 'at_warehouse' linked to the provider's transport bookings. Join with tenants for client_name. Calculate days_held. Order by days_held DESC.

    **`getOutboundShipmentHistory()`:**
    Returns `{ id: string; destination: string; status: string; pickup_count: number; created_at: string }[]`.
    Query outbound_shipments created by this provider (last 30 days), with count of linked transport_bookings.

    - Cards: "Awaiting Collection", "In Transit", "At Warehouse", "Completed (30d)"
