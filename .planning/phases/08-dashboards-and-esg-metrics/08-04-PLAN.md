  <action>
    Create `apps/web/app/(ops)/dashboard/actions.ts` with `'use server'` directive.

    All functions use raw `db` (no RLS context) — these are reco-admin cross-tenant queries. Import `db` from `@repo/db` and `sql` from `drizzle-orm`.

    **`getPickupStatusSummary(tenantFilter?: string)`:**
    Returns `{ status: string, count: number }[]`.
    Raw SQL:
    ```sql
    SELECT status, COUNT(*)::int AS count
    FROM pickups
    WHERE voided = false
    GROUP BY status
    ORDER BY CASE status
      WHEN 'submitted' THEN 1
      WHEN 'confirmed' THEN 2
      WHEN 'transport_booked' THEN 3
      WHEN 'picked_up' THEN 4
      WHEN 'at_warehouse' THEN 5
      WHEN 'in_outbound_shipment' THEN 6
      WHEN 'in_transit' THEN 7
      WHEN 'delivered' THEN 8
      WHEN 'intake_registered' THEN 9
      ELSE 10
    END
    ```
    Add optional `AND tenant_id = ${tenantFilter}` when provided.

    **`getConsolidationAgeing(tenantFilter?: string)`:**
    Returns `{ pickup_id: string, reference: string, client_name: string, pallet_count: number, arrival_date: string, days_held: number }[]`.
    Raw SQL joining pickups (status = 'at_warehouse') with tenants for client name. Calculate `days_held` as `EXTRACT(EPOCH FROM NOW() - arrival_date) / 86400`. Add optional tenant filter. Order by days_held descending.

    **`getPrisonPipeline(tenantFilter?: string)`:**
    Returns `{ facility_name: string, awaiting: number, processing: number, ready: number, shipped: number }[]`.
    Raw SQL joining intake_records + processing_reports + outbound_dispatches grouped by prison_facility_id, joined with prison_facilities for name. Use CASE/COUNT aggregation for stage counts. Filter `voided = false`. Add optional tenant filter.

    **`getRevenueSummary(tenantFilter?: string)`:**
    Returns `{ total_invoiced_eur: number, total_paid_eur: number, total_uninvoiced_eur: number, record_count: number }`.
    Raw SQL on financial_records:
    ```sql
    SELECT
      COALESCE(SUM(estimated_invoice_amount_eur) FILTER (WHERE invoice_status = 'invoiced'), 0) AS total_invoiced_eur,
      COALESCE(SUM(estimated_invoice_amount_eur) FILTER (WHERE invoice_status = 'paid'), 0) AS total_paid_eur,
      COALESCE(SUM(estimated_invoice_amount_eur) FILTER (WHERE invoice_status = 'not_invoiced'), 0) AS total_uninvoiced_eur,
      COUNT(*)::int AS record_count
    FROM financial_records
    ```
    Add optional tenant filter via join on intake_records.tenant_id.

    **`getDashboardTenants()`:**
    Returns `{ id: string, name: string }[]` from tenants table ordered by name.

    Cast all numeric SQL results to `Number()`. Handle null/empty results gracefully.
