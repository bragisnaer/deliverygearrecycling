    **`getEsgData(tenantFilter?: string, dateFrom?: string, dateTo?: string)`:**
    Returns `{ materials: MaterialWeightRow[], reuseRate: number, co2: { value_kg: number | null, formula_pending: boolean }, totalItems: number }`.

    Implementation:
    1. Build temporal composition join via raw SQL using Drizzle `sql` template tag (raw db, no RLS — admin query):
       ```sql
       SELECT
         il.actual_quantity,
         pm.weight_grams,
         ml.name AS material_name
       FROM intake_lines il
       JOIN intake_records ir ON ir.id = il.intake_record_id
       JOIN product_materials pm ON pm.product_id = il.product_id
         AND pm.effective_from <= ir.delivery_date
         AND (pm.effective_to IS NULL OR pm.effective_to > ir.delivery_date)
       JOIN material_library ml ON ml.id = pm.material_library_id
       WHERE ir.voided = false
       ```
       Add optional `AND ir.tenant_id = ${tenantFilter}` when tenantFilter provided.
       Add optional `AND ir.delivery_date >= ${dateFrom}` / `AND ir.delivery_date <= ${dateTo}` when date filters provided.

    2. Pass query results to `sumMaterialWeights()` from `@/lib/esg-calculator`.
    3. Call `calculateCO2Avoided(materials, null)` — stub.
    4. Calculate `totalItems` as sum of `actual_quantity` from query results.

    **`getProcessingStreamCounts(tenantFilter?: string)`:**
    Returns `{ reuse_qty: number, total_qty: number }`.

    Implementation via raw SQL:
    ```sql
    SELECT
      COALESCE(SUM(prl.quantity) FILTER (WHERE p.processing_stream = 'reuse'), 0) AS reuse_qty,
      COALESCE(SUM(prl.quantity), 0) AS total_qty
    FROM processing_report_lines prl
    JOIN processing_reports pr ON pr.id = prl.processing_report_id
    JOIN products p ON p.id = pr.product_id
    WHERE pr.voided = false
    ```
    Add optional `AND pr.tenant_id = ${tenantFilter}`.

    **`getEsgTenants()`:** Returns `{ id: string, name: string }[]` — list of all tenants for the filter dropdown. Query tenants table via raw db.

    Import `db` from `@repo/db` and `sql` from `drizzle-orm`. Import `sumMaterialWeights`, `calculateReuseRate`, `calculateCO2Avoided` from `@/lib/esg-calculator`. Cast numeric results from SQL (weight_grams, actual_quantity) to numbers using `Number()` before passing to pure functions.
