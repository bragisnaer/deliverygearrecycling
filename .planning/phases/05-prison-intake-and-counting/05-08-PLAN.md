    **Add to `apps/web/app/(ops)/intake/actions.ts`:**

    `getDiscrepancyByCountry()`:
    - requireRecoAdmin()
    - Single aggregate SQL via Drizzle `sql` template tag (Pitfall 5: no N+1):
    ```sql
    SELECT
      ir.origin_market AS country,
      COUNT(*) AS total_deliveries,
      SUM(CASE WHEN ir.discrepancy_flagged THEN 1 ELSE 0 END) AS flagged_count,
      ROUND(100.0 * SUM(CASE WHEN ir.discrepancy_flagged THEN 1 ELSE 0 END) / NULLIF(COUNT(*), 0), 1) AS discrepancy_rate_pct
    FROM intake_records ir
    WHERE ir.delivered_at >= NOW() - INTERVAL '6 months'
    GROUP BY ir.origin_market
    ORDER BY discrepancy_rate_pct DESC
    ```
    - Return: `{ country: string, total_deliveries: number, flagged_count: number, discrepancy_rate_pct: number }[]`

    `getDiscrepancyByProduct()`:
    - requireRecoAdmin()
    - Aggregate over intake_lines JOIN products:
    ```sql
    SELECT
      p.name AS product_name,
      COUNT(*) AS total_lines,
      SUM(CASE WHEN il.discrepancy_pct IS NOT NULL AND il.discrepancy_pct > 0 THEN 1 ELSE 0 END) AS flagged_lines,
      ROUND(100.0 * SUM(CASE WHEN il.discrepancy_pct IS NOT NULL AND il.discrepancy_pct > {threshold} THEN 1 ELSE 0 END) / NULLIF(COUNT(*), 0), 1) AS discrepancy_rate_pct
    FROM intake_lines il
    JOIN intake_records ir ON ir.id = il.intake_record_id
    JOIN products p ON p.id = il.product_id
    WHERE ir.delivered_at >= NOW() - INTERVAL '6 months'
    GROUP BY p.name
    ORDER BY discrepancy_rate_pct DESC
    ```

    `getDiscrepancyByFacility()`:
    - requireRecoAdmin()
    - Aggregate over intake_records JOIN prison_facilities:
    ```sql
    SELECT
      pf.name AS facility_name,
      COUNT(*) AS total_deliveries,
      SUM(CASE WHEN ir.discrepancy_flagged THEN 1 ELSE 0 END) AS flagged_count,
      ROUND(100.0 * SUM(CASE WHEN ir.discrepancy_flagged THEN 1 ELSE 0 END) / NULLIF(COUNT(*), 0), 1) AS discrepancy_rate_pct
    FROM intake_records ir
    JOIN prison_facilities pf ON pf.id = ir.prison_facility_id
    WHERE ir.delivered_at >= NOW() - INTERVAL '6 months'
    GROUP BY pf.name
    ORDER BY discrepancy_rate_pct DESC
    ```

    `getMonthlyDiscrepancyByCountry(country: string)`:
    - For persistent problem flag: fetch monthly rates for a specific country over last 6 months
    ```sql
    SELECT
      date_trunc('month', ir.delivered_at) AS month,
      ROUND(100.0 * SUM(CASE WHEN ir.discrepancy_flagged THEN 1 ELSE 0 END) / NULLIF(COUNT(*), 0), 1) AS rate
    FROM intake_records ir
    WHERE ir.origin_market = {country}
      AND ir.delivered_at >= NOW() - INTERVAL '6 months'
    GROUP BY date_trunc('month', ir.delivered_at)
    ORDER BY month ASC
    ```
    - After fetching: call `isPersistentProblemMarket(rates)` to compute the flag
    - Return: `{ month: string, rate: number }[]` + `persistentFlag: boolean`

    All queries use `withRLSContext(user, (tx) => tx.execute(sql`...`))`. Read threshold from system_settings.
