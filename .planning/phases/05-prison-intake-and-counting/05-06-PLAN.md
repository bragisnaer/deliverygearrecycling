    - Insert notification for BOTH prison staff (current user) and reco-admin: type='quarantine_flagged', non-blocking
    - If no override is present and quarantine_flagged: return error, do not insert

    - Visual: red destructive banner with role="alert" for screen reader announcement
    - Submit button text changes to "Karantaene — afventer godkendelse" when blocked

    Note: The quarantine block on the client is cosmetic — the server-side check in submitIntake is the real enforcement. Both must agree.
