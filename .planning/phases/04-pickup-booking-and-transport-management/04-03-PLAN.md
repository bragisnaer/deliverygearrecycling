**Server Actions (`apps/web/app/(ops)/pickups/actions.ts`):**

1. `'use server'` at top
2. Import auth, db, pickups, pickupLines, locations, products, withRLSContext, eq, and, revalidatePath, z
3. `requireRecoAdmin()` helper (same pattern as products/actions.ts)
4. `confirmPickup(pickupId: string)`:
   - Validate pickup status is 'submitted'
   - withRLSContext: UPDATE pickups SET status='confirmed', confirmed_date=now(), updated_at=now() WHERE id=pickupId
   - revalidatePath('/pickups')
   - Return `{ success: true }`
5. `cancelPickup(pickupId: string, reason: string)`:
   - Validate pickup status is NOT 'delivered', 'intake_registered', or 'cancelled'
   - withRLSContext: UPDATE pickups SET status='cancelled', cancellation_reason=reason, cancelled_at=now(), cancelled_by=user.sub, updated_at=now() WHERE id=pickupId
   - revalidatePath('/pickups')
   - Return `{ success: true }`
6. `updatePickupStatus(pickupId: string, newStatus: string)`:
   - Validate allowed transitions (submitted->confirmed, confirmed->transport_booked, etc.)
   - withRLSContext: UPDATE pickups SET status=newStatus, updated_at=now() WHERE id=pickupId
   - revalidatePath('/pickups')
