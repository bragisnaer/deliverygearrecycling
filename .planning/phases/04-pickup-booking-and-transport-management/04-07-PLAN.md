<objective>
Build outbound shipment creation from warehouse inventory. reco-admin selects held pickups via checkboxes, enters destination prison and warehouse-to-prison cost. The system proposes pro-rata cost allocation by pallet count; reco-admin can override individual allocations. On confirmation, outbound_shipments + outbound_shipment_pickups rows are created and pickup statuses advance. Marking the shipment delivered cascades status to all linked pickups.

Purpose: Completes the consolidation transport workflow — the second leg of the two-leg cost model.
Output: Outbound shipment creation page, pro-rata allocation logic with tests, delivery cascade action.
</objective>

Add to the existing `apps/web/app/(ops)/transport/outbound/actions.ts`:

1. **Pure function** `calculateProRataAllocation(totalCost: string, pickups: Array<{pickupId: string, palletCount: number}>)`:
   - Total pallets = SUM of all pallet counts
   - For each pickup: `allocated = (palletCount / totalPallets) * parseFloat(totalCost)`
   - Round each to 4 decimal places
   - Distribute rounding remainder to the last pickup so allocations sum to exactly totalCost
   - Return `Array<{pickupId: string, palletCount: number, allocatedCostEur: string}>` where allocatedCostEur is toFixed(4)
   - Export this function for testing

2. `outboundShipmentSchema` Zod schema:
   - `prison_facility_id`: z.string().uuid()
   - `transport_cost_warehouse_to_prison_eur`: z.string().regex(/^\d+(\.\d{1,4})?$/)
   - `pickup_allocations`: z.array(z.object({ pickup_id: z.string().uuid(), pallet_count: z.number().int().min(1), allocated_cost_eur: z.string() })).min(1)

3. `createOutboundShipment(formData: FormData)`:
   - requireRecoAdmin() (or requireRecoAdminOrTransport for consolidation providers)
   - Parse + validate
   - Validate sum of allocated_cost_eur equals transport_cost_warehouse_to_prison_eur (with epsilon tolerance of 0.01)
   - Compute total_pallet_count = SUM of pickup pallet_counts
   - withRLSContext:
     - INSERT outboundShipments row
     - INSERT outboundShipmentPickups rows for each pickup with pallet_count and allocated_cost_eur
     - UPDATE all selected pickups: SET status='in_outbound_shipment', updated_at=now()
   - revalidatePath('/transport/outbound')
   - Return `{ success: true, shipmentId }`

4. `markOutboundDelivered(shipmentId: string)`:
   - requireRecoAdminOrTransport()
   - withRLSContext:
     - UPDATE outbound_shipments SET status='delivered', delivered_at=now(), updated_at=now() WHERE id=shipmentId
     - SELECT all pickup_ids from outbound_shipment_pickups WHERE outbound_shipment_id=shipmentId
     - UPDATE pickups SET status='delivered', updated_at=now() WHERE id IN (pickup_ids)
     - Also UPDATE related transport_bookings if needed
   - revalidatePath('/transport/outbound')
   - Return `{ success: true }`
